      BitVec

      @memory_only
struct BitVec

      A growable bitfield.

      This uses one bit per bool for storage.

      The bits are stored in Self.WORD_DTYPE words. This is optimized for compactness and speed.

      Aliases

      • WORD_DTYPE = DType.uint64 if (is_gpu() ^ True) else DType.uint32
      • WORD_BYTEWIDTH = (DType.uint64 if (is_gpu() ^ True) else DType.uint32.bit_width() // 8)
      • WORD = Scalar[DType.uint64 if (is_gpu() ^ True) else DType.uint32]
